Help & FAQ

Frequently Asked Questions


üîç How do I search for vehicles?

Go to the Vehicles page and use the search bar to find vehicles by license plate, make, model, or owner name. The search updates results in real-time as you type.

üé® What do the badge colors mean?

Gray: Silver Alert (missing elderly person)
Orange: Amber Alert (child abduction)
Red: Stolen Vehicle
Purple: Arrest Warrant (owner has active warrant)

üìä What information is displayed?

Each vehicle record shows license plate, reason for alert, make/model, year, color, owner information, and last update date. Click any row to see detailed information in a popup.

üîÑ How often is the data updated?

This web application displays read-only data. All updates are managed by the desktop application used by authorized personnel. The web app shows the current state of the database.

üîê How does authentication work?

For demonstration purposes, the login accepts any non-empty username and password. In production, this would be integrated with the department's authentication system.

üì± Is this mobile-friendly?

Yes! The interface adapts to mobile devices. On smaller screens, the sidebar navigation is hidden to maximize viewing space.
